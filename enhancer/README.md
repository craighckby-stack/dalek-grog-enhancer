# Sovereign Enhancer

Context-aware continuous evolution engine.

## Setup

```bash
npm install
npm run dev
```

## Configuration

Set your credentials in the UI:
1. GitHub Token (with repo access)
2. Cerebras API Key
3. Target repository (user/repo format)

## Features

- README-priority context priming
- Project-wide awareness
- Automatic mutation commits
- Telemetry dashboard
- Resume mode for interrupted sessions
